JavaScript Methods - document.write();

A003JS

As you write your JavaScript code, please remember that JavaScript is not forgiving as HTML. If you code something wrong, your webpage generates an error. Be patient and keep these things in mind when coding JavaScript.

  • JavaScript is case sensetive
  • JavaScript is white-space sensetive (enters matter)
  • JavaScript is sensetive to apastrophes (or single quotes)
  • JavaScript lines should end with a semi-colon

The browser window contains other objects, such as the document object. You can refer to the document object by using "window.object." The following is a method (action) of the document object. This method is called by using the following code:

window.document.write();

Here's the code:

        
            window.document.write("<p class='highlight'> I am using JavaScript to write to the page. This text has been made using JavaScript window.document.write() </p>");
