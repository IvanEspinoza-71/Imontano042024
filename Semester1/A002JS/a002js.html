JavaScript Methods - window.alert()

A002JS

JavaScript is a scripting language that uses objects. For example, your browser window is an object. We can refer to this object as via JavaScript by using "window."

The following is a method (action) of the window object. This method is called by using the following code:

window.alert("Welcome to my Webpage!")

Here's the code used to create the window.alert:

        
        // Use code to alert your user
        // Use this alert method of the window object

        window.alert("Welcome to my Webpage!");
